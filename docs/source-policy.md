# 来源政策

`last7days-rent-skill` 的目标是给用户一份可溯源、可联系的候选短名单，而不是承诺所有房源都已经验真。来源策略优先考虑可访问性、字段结构、联系方式可行动性和后续核验动作。

## 全渠道矩阵

这里的“全渠道”是产品概念，不是任意抓取互联网内容。默认只覆盖合规公开渠道和用户授权输入。

| 渠道 | 示例 | 自动公开发现 | 用户授权导入 | 默认处理 |
| --- | --- | --- | --- | --- |
| 公开租房平台 | 贝壳/链家、58/安居客、房天下、自如、我爱我家等公开入口 | 是 | 是 | 生成公开搜索 query，打开可访问详情页，遇到登录/验证码/安全中心则记录 blocked |
| 品牌公寓公开页 | 乐乎、泊寓、城家、有巢等公开页面 | 是 | 是 | 作为过渡/备选房源，预约、手机号验证码和 App 内流程留给用户 |
| 公开转租社区 | 豆瓣公开小组、Wellcee、公开帖子/文章 | 是 | 是 | 默认更严格标风险；公开正文可整理，私信/登录后内容不采集 |
| 用户授权材料 | 用户给的链接、截图、复制文本、本地整理材料 | 否 | 是 | 按 evidence 导入并记录 provenance；不自动读取聊天记录或私域应用 |
| 私域或登录后内容 | 微信群、朋友圈、公司群、校友群、登录后页面、验证码后内容 | 否 | 仅人工摘要后导入 | 不自动抓取、不绕登录、不保存 cookie/token |

内部实现仍可以用优先级标记：公开且结构稳定的来源优先执行；有价值但噪声较高的来源作为补充；私域和登录后内容不进入自动采集。

## 当前优先来源

- 贝壳 / 链家 / Ke：主流公开列表，适合抽取 URL、平台 ID、标题、价格、面积、户型、位置、维护时间和平台联系入口；详情页遇到安全中心、登录或验证码时必须记录 blocked，不绕过。
- 链家移动列表：可作为北京京东 POC 的公开列表卡片来源，优先提取标题、价格、面积、户型、朝向、地铁距离和平台入口。
- Wellcee：详情页如包含 `RealEstateListing` JSON-LD 或页面可见结构字段，可解析为单源候选；页面公开展示的联系方式或原帖联系说明应保留。
- 房天下：可作为补充公开来源，但字段结构需独立适配。
- 豆瓣公开小组：只采集公开帖正文，默认 L0；字段足够且页面打开后可进入 L1。
- 乐乎 / 泊寓 / 品牌公寓公开页：作为过渡房源来源，满足 L1 条件后可进入候选；约看、手机号验证码或 App 内流程必须留给用户人工处理。
- 官方核验入口：只作为 `VerificationEvidence`，不作为高召回来源。
- NestHub：只作为北京京东 POC 的先验知识和 query/risk seed，不作为实时房源事实。

## 后续可评估来源

- 小红书、微博、公众号公开文章等高噪声公开内容，只能在明确产品策略和风险标签后加入公开发现。
- 微信群、朋友圈、公司群、校友群等私域内容不做自动采集；如果用户主动复制文本或截图导入，只能按用户授权材料处理。
- 任何需要登录、验证码、cookie、token 或批量反机器人绕过的来源，都不能纳入自动全渠道。

## 联系方式政策

可公开访问页面展示的联系方式是房源可用性的核心字段，应作为 `contact_methods` 保留。平台只提供站内联系时，报告应展示原始 URL 和联系动作说明。

核心短名单里的房源必须至少有一种可行动联系路径：

- 平台联系入口。
- 电话。
- 微信、QQ、飞书。
- 邮箱。
- 原帖联系说明。
- 用户授权导入联系人。

没有联系方式且没有可打开联系入口的候选，只能作为低价值线索或 warning，不能当成可行动房源推荐。

## 用户授权导入

用户可以显式提供公开 URL、本地文本或剪贴文本。导入内容按原文结构化，保留用户提供的联系方式，并记录 provenance。用户授权导入的候选最高为 L1，除非用户后续反馈确认。

不允许自动读取用户聊天记录、私域群、朋友圈、公司群或校友群。不得保存 cookie、token、secret 或登录态凭证。

## WebSearch discovery

WebSearch 可以用于发现入口和来源分类，但搜索结果 snippet 不能直接转成正式房源。snippet 中的价格、发布时间、地址和联系方式只能作为弱线索，必须由详情页、平台页面或用户授权输入复核。

## 验证边界

- L1：单源结构化，有真实 URL 或用户授权导入证据，并有可行动联系路径。
- L2：多源佐证或重复出现，关键字段和联系路径基本一致。
- L3：用户联系确认或明确反馈仍在租、可看房。

平台编号、官方核验码、多源重复都不能单独把候选房源提升为 L3。
