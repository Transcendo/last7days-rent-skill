## 实现要求

- Source registry 必须记录：
  - `source_id`
  - domain
  - access level A/B/C/D
  - allowed output type
  - default trust level
  - requires authorization
  - can promote to listing
- 贝壳/链家/Ke：
  - 解析公开列表卡片。
  - 维护时间只能作为弱 freshness。
- Wellcee：
  - 优先解析 `RealEstateListing` JSON-LD。
  - `datePosted` 标记待核验。
- 房天下：
  - 独立 selector，不复用贝壳结构。
- 官方核验：
  - 只输出 `VerificationEvidence`。
  - 不作为高召回来源。
- Source registry 必须显式标记非 MVP 来源为 disabled/non_mvp：
  - 自如。
  - 我爱我家。
  - 58/安居客。
  - 豆瓣。
  - 小红书/微博/公众号。
  - 私域/用户授权导入。
  - WebSearch discovery。

## 不做的事

- 不实现 WebSearch discovery。
- 不实现用户授权导入。
- 不处理私域内容。
- 不抓豆瓣。
- 不抓 58/安居客详情。
- 不绕自如/58/安居客验证码。
- 不抓小红书/微博登录态内容。
- 不要求用户提供 cookie。
- 不自动采集微信群、朋友圈、公司群。

## 验收标准

- 每个 adapter 有 fixture 测试。
- Source registry 中 P1/P2/私域/WebSearch 来源处于 disabled 或 non_mvp 状态。
- 非 MVP 来源无法进入 MVP pipeline。
- source 失败返回 warning，不阻断 pipeline。
- 输出中不含未经脱敏的联系方式。
