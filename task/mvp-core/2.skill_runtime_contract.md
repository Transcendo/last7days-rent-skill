## 实现要求

- `SKILL.md` 必须说明触发条件：
  - 用户想 7 天完成租房。
  - 用户要求找近 7 天房源。
  - 用户要求基于公司/办公点/预算/通勤找房。
  - 用户要求导入房源线索并脱敏。
- CLI 至少支持：
  - `profile init`
  - `profile show --redacted`
  - `profile refine`
  - `search`
  - `feedback`
  - `report`
  - `--help`
- 默认本地目录：
  - `~/.nesthub-rent/profile.json`
  - `~/.nesthub-rent/profile.md`
  - `~/.nesthub-rent/feedback.jsonl`
  - `~/.nesthub-rent/reports/`
- Runtime 必须支持 fixture/offline 模式，方便测试和最终验收。
- `SKILL.md` 必须禁止：
  - 绕验证码。
  - 抓取私域。
  - 暴露私人联系方式。
  - 把 L1 写成已验真。

## 不做的事

- 不实现完整 profile 问答。
- 不实现 source adapter。
- 不实现排序和报告。
- 不写真实用户数据到 repo。

- `python skills/last7days-rent/scripts/last7days_rent.py --help` 可运行。
- `SKILL.md` 包含 slogan 和隐私边界。
- CLI 默认路径不在 repo 内保存用户 profile。
